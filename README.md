PrimeDwell - House Rental Web Application

Overview

PrimeDwell is a role-based house rental web application with dedicated admin, owner, and tenant dashboards. It allows users to browse properties, owners to list properties, and admins to manage the platform.


 Features

 1.Tenant
o	Register & Login
o	Browse properties
o	Send requests
o	Contact owner

  1. Owner
o	Add property with images
o	Manage property listings
o	View tenant requests

   2. Admin

o	Manage users
o	Manage properties
o	View contact requests
o	Dashboard overview



 Technologies Used

PHP
MySQL
HTML, CSS, JavaScript
XAMPP (Local Server)

Chat system

The project includes a messaging system where tenants and owners can communicate directly. 
Messages are stored in the database and updated dynamically to provide smooth interaction between users.


 How to Run the Project

1. Install XAMPP
2. Copy project folder to:
   C:\xampp\htdocs\
3. Start Apache & MySQL
4. Import database in phpMyAdmin
5. Run in browser:
   http://localhost/website


note-
This project runs on a local server environment.
